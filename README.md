# Portfólio

Projeto desenvolvido com o objetivo de mostrar meus conhecimentos em desenvolvimento web listando alguns projetos pessoais que desenvolvi.

## Desenvolvido utilizando

    - SCSS (SASS)
    - JavaScript
    - Webpack
      - Autoprefixer
      - PostCSS
      - PurgeCSS
